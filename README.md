# Agentic Engineering

Agentic Engineering is a collection of experimental projects that explore the capabilities and workflows of modern AI coding agents and developer tools. This repository serves as a comparative showcase of how different agentic tools—Cursor, GitHub Copilot, and Claude—can be leveraged for software development.

## Overview

The goal of this repository is to:
- **Compare** the strengths and limitations of various agentic coding tools.
- **Demonstrate** practical use cases for AI-assisted software development.
- **Document** workflows, prompts, and outcomes for each tool.
- **Provide** a reference for developers interested in agentic engineering.

## Project Structure


- **cursor-project:** Code and experiments using Cursor AI.
- **copilot-project:** Solutions powered by GitHub Copilot.
- **claude-project:** Code generated and refined with Anthropic’s Claude.

Each sub-project contains its own documentation and instructions.
